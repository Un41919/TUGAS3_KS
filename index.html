Pendahuluan

Apa itu iterasi?

Dalam dunia pemrograman dan komputasi, iterasi adalah proses mengulang serangkaian instruksi atau operasi sebanyak beberapa kali, baik dengan jumlah pengulangan yang telah diketahui atau berdasarkan suatu kondisi tertentu (Franke, 1988).

Mengapa penting?

Iterasi memungkinkan kita untuk:

  • Menyederhanakan pekerjaan berulang.
  • Memproses data dalam jumlah besar.
  • Menghindari penulisan kode yang panjang dan berulang.

Konsep Iterasi

Pengertian

Iterasi adalah struktur kontrol dalam pemrograman yang digunakan untuk mengeksekusi blok kode secara berulang hingga kondisi tertentu terpenuhi (Kulkarni, 2022).

Fungsi utama iterasi

  • Efisiensi dalam coding.
  • Mengotomatisasi proses berulang.
  • Menghindari kesalahan manual.

Beberapa Peran Iterasi dalam Komputasi Statistika

✅ Untuk pengulangan perhitungan statistik, seperti rata-rata bergerak, uji berulang, atau evaluasi model.

✅ Digunakan dalam simulasi statistik, seperti bootstrap atau Monte Carlo.

✅ Berguna dalam memproses data besar secara otomatis, baris demi baris atau dalam batch.

For Loop

Pengertian:
For loop digunakan saat jumlah pengulangan sudah diketahui.

Struktur kode:

for (i in 1:5) {
  print(i)
}

Kelebihan:
- Ringkas dan mudah dibaca.
- Cocok untuk struktur data seperti vector dan list.
- Ideal untuk pengolahan data berukuran tetap.

While Loop

Pengertian:
While loop digunakan ketika pengulangan didasarkan pada kondisi logika.

Struktur kode:

i <- 1
while (i <= 5) {
  print(i)
  i <- i + 1
}

Kelebihan:
- Sangat fleksibel.
- Cocok untuk input dinamis, validasi, atau proses simulasi dengan kondisi.

For VS While

Aspek For Loop While Loop
Kontrol Jumlah iterasi tetap Berdasarkan kondisi logika
Kapan digunakan Saat iterasi pasti (misal: 1–100) Saat belum tahu kapan selesai
Risiko infinite loop Rendah Tinggi jika kondisi tak dikontrol
Fleksibilitas Terbatas karena harus menetapkan jumlah percobaan Lebih fleksibel karena menunggu kondisi benar
Kelebihan Dapat diatur dengan batasan percobaan yang pasti Lebih cocok untuk kondisi yang tidak pasti
Contoh penggunaan Traverse vector, matriks Validasi input, simulasi dinamis
Contoh kasus Pengulangan dengan jumlah iterasi tetap Menunggu input valid dari user

Contoh Penggunaan

Kasus 1: Jumlahkan angka 1 sampai 5

1. Menggunakan For

total <- 0
for (i in 1:5) {
  total <- total + i
}
print(total)

2. Menggunakan While

total <- 0
i <- 1
while (i <= 5) {
  total <- total + i
  i <- i + 1
}
print(total)

✅ Sama-sama benar tapi While sedikit lebih panjang.

Kesimpulan Kasus 1:
Keduanya bisa menyelesaikan masalah yang sama, namun for lebih disarankan di kasus seperti ini karena lebih ringkas.

Kasus 2: Validasi input angka positif dari user

1. Menggunakan For

for (i in 1:100) {  # Asumsi maksimal 100 percobaan
  angka <- as.integer(readline("Masukkan bilangan positif: "))
  if (angka > 0) {
    break
  }
}
cat("Angka valid:", angka, "\n")

2. Menggunakan While

angka <- -1
while (angka <= 0) {
  angka <- as.integer(readline("Masukkan bilangan positif: "))
}
cat("Angka valid:", angka, "\n")

Kesimpulan Kasus 2:

  • Karena kita nggak tahu berapa kali user akan salah input.
  • Proses akan terus berjalan hingga input memenuhi kondisi yang benar.

Keimpulan

  • Iterasi adalah komponen penting dalam pemrograman dan komputasi statistika.

  • For cocok untuk perulangan dengan jumlah pasti dan terkontrol.

  • While ideal untuk proses berbasis kondisi dinamis.

  • Dalam statistika, iterasi berperan dalam:

    • Pengulangan perhitungan,
    • Simulasi statistik,
    • Analisis data besar,
    • Dan berbagai algoritma komputasi kompleks.
